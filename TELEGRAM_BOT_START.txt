🚀 БЫСТРЫЙ СТАРТ:

1️⃣  ⬜ Создайте Telegram бота (@BotFather → /newbot)
    📌 Сохраните BOT_TOKEN

2️⃣  ⬜ Получите свой Telegram ID
    (Найдите бота → /start → /myid)
    📌 Сохраните ваш ID

3️⃣  ⬜ Создайте конфиг для бота:
    cp telegram_bot/.env.example telegram_bot/.env
    
    Отредактируйте telegram_bot/.env:
    - TELEGRAM_BOT_TOKEN=<ваш_токен>
    - ADMIN_IDS=[<ваш_id>]
    - API_KEY=<придумайте_сложный_ключ>

4️⃣  ⬜ Создайте конфиг для фронтенда:
    cp .env.local.example .env.local
    
    Отредактируйте .env.local:
    - VITE_API_KEY=<тот_же_ключ>

5️⃣  ⬜ Установите Python зависимости:
    cd telegram_bot
    pip install -r requirements.txt
    cd ..

6️⃣  ⬜ Запустите API сервер (Терминал 1):
    cd telegram_bot
    python api_server.py
    
    Должно вывести: Running on http://127.0.0.1:5000/

7️⃣  ⬜ Запустите фронтенд (Терминал 2):
    npm run dev
    
    Откройте: http://localhost:3000

8️⃣  ⬜ Тестируйте систему:
    - Откройте http://localhost:3000/контакты
    - Заполните форму
    - Нажмите "Отправить запрос"
    - Проверьте Telegram ✅

📊 КАК ЭТО РАБОТАЕТ:

User Form
   ↓
   POST /api/inquiry
   ├─ name, phone, message
   ├─ X-API-Key header
   └─ JSON body
        ↓
   Flask API Server
   ├─ Проверяет API_KEY
   ├─ Валидирует данные
   └─ Отправляет в Telegram
        ↓
   Telegram Bot API
   ├─ Получает от нашего сервера
   └─ Доставляет в чаты админов/менеджеров
        ↓
   ✅ Админ получает уведомление в Telegram

❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

1. Проверьте что Python и pip установлены:
   python3 --version
   pip3 --version

2. Проверьте что порты свободны:
   - 3000 (фронтенд)
   - 5000 (API)

3. Проверьте конфигурацию:
   - API_KEY в обоих .env файлах должны совпадать
   - TELEGRAM_BOT_TOKEN правильный
   - Admin/Manager IDs правильные

4. Запустите тест:
   python telegram_bot/test_api.py

5. Прочитайте полную инструкцию:
   TELEGRAM_SETUP.md
